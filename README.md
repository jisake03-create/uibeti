# UIBETI Web App

可直接部署到 Vercel 的 React + Vite 项目。

## 本地运行

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```

## 部署到 Vercel

### 方式一：网页上传
1. 把整个项目文件夹压缩成 zip。
2. 打开 Vercel。
3. New Project → Import → Upload。
4. 上传这个项目。
5. 保持默认设置，点击 Deploy。

### 方式二：GitHub 部署
1. 把项目上传到 GitHub 仓库。
2. 在 Vercel 中导入该仓库。
3. Framework 识别为 Vite。
4. 点击 Deploy。
